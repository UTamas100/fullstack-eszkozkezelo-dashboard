# Fullstack Eszközkezelő Dashboard

Ez a projekt egy Angular 19 + PrimeNG frontendből és egy Node.js + MongoDB backendből áll.

## Indítás

### Backend:
```bash
cd backend
npm install
npm start
```

### Frontend:
```bash
cd frontend
npm install
ng serve
```

## API-k
```bash
GET /devices
POST /devices
DELETE /devices/:id
```
